Avatar头像 #

基础使用 #

传入img的src

<v-avatar></v-avatar>
<v-avatar src="https://cube.elemecdn.com/e/fd/0fc7d20532fdaf769a25683617711png.png"></v-avatar>

形状及尺寸 #

传入shape 及 size

  <v-avatar></v-avatar>
  <v-avatar size="medium"></v-avatar>
  <v-avatar size="small"></v-avatar>
  <v-avatar src="https://cube.elemecdn.com/e/fd/0fc7d20532fdaf769a25683617711png.png" shape="square"></v-avatar>
  <v-avatar src="https://cube.elemecdn.com/e/fd/0fc7d20532fdaf769a25683617711png.png" shape="square" size="medium"></v-avatar>
  <v-avatar src="https://cube.elemecdn.com/e/fd/0fc7d20532fdaf769a25683617711png.png" shape="square" size="small"></v-avatar>

<v-avatar @error="handleError"></v-avatar>
<v-avatar class="ml-20" src="https://cube.elemecdn.com/e/fd/0fc7d20532fdaf769a25683617711png.png"></v-avatar>

<script setup>
  const handleError = () => {}
</script>
